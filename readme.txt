== Description ==

Плагин позволяет настроить отображение ткани для woo-commerce и вывести ее на странице товара в модальном окне.
Важно, чтобы название тканей совпадало с названием цветов в вариативном товаре woo-commerce.


== Installation ==


1. Загрузите и распакуйте файлы плагина в директорию `/wp-content/plugins/` или загрузите и установите плагин через стандартный интерфейс добавления плагинов в  WordPress.
2. Активируйте плагин на странице установленных плагинов WordPress
3. В меню появится пункт "Выбор ткани" - щелкните на него для настроек
4. Каждая ткань имеет свою закладку. Заполните названия тканей и нажмите кнопку  "Сохранить изменения" внизу.
